TypeScriptUsers ist ein offenes Verzeichnis von TypeScript-Entwicklern in Deutschland.

Unser Ziel ist es, die Bekanntheit von TypeScript zu fördern, indem wir TypeScript-Entwickler untereinander vernetzen

TypeScript

TypeScript ist ein Superset von JavaScript und erweitert dieses um statische Typen und Konzepte der Objektorientierung, wie

Klassen,
Module,
Interfaces
und statische Typprüfungen.

TypeScript kompiliert zu JavaScript und kann daher auf jeder Zielplattform ausgeführt werden.

Über Uns

Die Seite TypeScriptUsers.de ist ein Projekt der TypeScript User Group Deutschland e.V.

Wir wollen die Bekanntheit von TypeScript fördern, indem wir der deutschen TypeScipt-Community die Möglichkeit bieten sich zu vernetzen und über TypeScript auszutauschen.

Neben unserem offenen Entwicklerverzeichnis wollen wir als User Group auch regelmäßige Treffen organisieren.
